# 🛍️ ShopAdmin - React Admin Dashboard

A modern and responsive e-commerce admin dashboard built with **React.js**, **Vite**, **React Router**, **Tailwind CSS**, and **Lucide React**. This project demonstrates a complete frontend workflow for managing products, orders, carts, checkout, and user profiles with a clean and reusable component architecture.

## ✨ Features

- 🔐 Login & Registration UI
- 📊 Dashboard with statistics cards
- 📦 Product listing page
- 🔍 Product details page
- 🛒 Shopping cart management
- 💳 Checkout page
- 📋 Order management
- 👤 User profile page
- 📱 Fully responsive design
- 🎨 Modern UI inspired by professional admin dashboards
- ⚡ Built using reusable React components

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/shopadmin.git
```

Move into the project

```bash
cd shopadmin
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
